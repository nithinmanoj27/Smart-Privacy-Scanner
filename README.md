#  Smart Privacy Scanner

An Android application that helps users understand the privacy risks associated with installed applications by analyzing permissions, trackers, and external privacy information.

The application retrieves privacy data from Exodus Privacy, evaluates applications using a heuristic-based risk scoring model, and presents understandable privacy insights to users.

---

##  Features

-  Analyze installed Android applications.
-  Detect third-party trackers.
-  Analyze dangerous permissions.
-  Generate privacy risk scores.
-  Fetch privacy information using Jsoup web scraping.
-  Explain trackers and permissions in a user-friendly way.
